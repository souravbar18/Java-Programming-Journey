# ☕ Basic Java Programming Journey

Welcome to my Java learning repository! This project documents my transition from **Swift/SwiftUI** to **Java**, focusing on building a strong foundation in Object-Oriented Programming (OOP) and logic-based problem solving.

## 🚀 About This Project
I am a self-taught programmer from Bangladesh. After exploring the Apple ecosystem (Swift), I decided to master Java to align with university standards and industry demands. This repository contains my daily practice codes and logic-building projects developed with the help of **Gemini AI**.

## 📂 Projects Included

### 1. 🔢 Arithmetic Calculator
A simple tool that handles basic math operations (Addition, Subtraction, Multiplication, Division) with input validation to prevent division by zero.

### 🗓️ 2. Leap Year Checker
A logic-heavy script that determines if a year is a Leap Year based on the standard Gregorian calendar rules (handling the 100/400-year exceptions).

### ⚖️ 3. Even-Odd Checker
A fundamental program using the modulo operator to distinguish between even and odd integers.

### 🎓 4. Grade Calculator
A school grading system implementation using `if-else` ladders to categorize marks into grades (A+, A, B, etc.).

### 🌡️ 5. Temperature Converter
Converts Celsius to Fahrenheit, demonstrating the implementation of mathematical formulas in code.

### 🔤 6. Vowel & Consonant Checker
A string-based program that identifies alphabets and handles invalid inputs using character comparison logic.
